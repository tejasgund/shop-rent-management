# Shop Rent Management System

A complete full-stack application for managing shop rentals, tenants, payments, and billing.

## Features

- **Multi-user Role System**: Admin and Tenant portals
- **Complete Tenant Management**: Create, edit, delete tenants with profile management
- **Payment Processing**: Manual payment entry with automatic ledger updates
- **Bill Management**: Handle electricity, water, maintenance bills
- **Account Ledger**: Automatic balance calculation and transaction history
- **Reports**: Generate various reports in CSV/PDF format
- **Responsive UI**: Works on desktop and mobile devices
- **Docker Support**: Easy deployment with Docker Compose

## Tech Stack

- **Backend**: FastAPI, SQLAlchemy, MySQL
- **Frontend**: React.js, Material-UI
- **Database**: MySQL 8.0
- **Authentication**: JWT
- **Deployment**: Docker
